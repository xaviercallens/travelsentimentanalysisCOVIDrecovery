Load_Model sert de module a importer pour utiliser un modele entrainé. IL NE FAUT PAS MODIFIER Load_Model,juste l'importer.
Le fichier "Exemple pour Load_Model" montre comment utiliser les fonctions de Load_Model.

Si il manque le module 'bert', il faut installer le module 'bert-for-tf2'.


Un modele entrainé se presente sous la forme d'un dossier. Il faudra placer ce dossier dans le meme dossier que celui que vous utilisez, ou alors modifier le chemin d'acces pour pouvoir l'utiliser.
